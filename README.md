# FleetFixr

**FleetFixr** is a full-stack **repair and maintenance CRUD application** designed to track and manage fleet repairs, maintenance tasks, and service records. This project includes both a backend service and a frontend user interface to help you build or expand a fleet management system. :contentReference[oaicite:0]{index=0}

---

##  Features

-  **Create, Read, Update, Delete (CRUD)** functionality for maintenance records  
-  Backend API for managing fleet services  
-  Frontend client for interacting with fleet data  
-  Modular codebase separated into service and frontend directories  
-  Great starting point to build a production-ready fleet management platform
